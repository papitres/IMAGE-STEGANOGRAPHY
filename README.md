# Image-Based Steganography System (Final Year Project)

A secure Java application that hides encrypted text messages inside images using LSB (Least Significant Bit) technique.

## Features
- Hide text messages inside PNG/JPG images
- AES encryption + passkey protection
- Decoy/fake message feature
- Extract hidden message with correct password only

## Tech Stack
- Java Swing (GUI)
- AES Encryption
- Steganography (LSB Algorithm)
